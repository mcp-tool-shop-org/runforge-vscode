# RunForge

Push-button ML training from VS Code — presets, indexed runs, and artifact inspection without leaving your editor.

## Key Features

- **One-click training** with Standard and High Quality presets
- **Model selection** — Logistic Regression, Random Forest, Linear SVC
- **Training profiles** — default, fast, thorough with hyperparameter control
- **Run browser** — indexed runs with metadata, metrics, and artifacts
- **Feature importance** and linear coefficient visualization
- **Interpretability index** for model transparency scoring

## Install / Quick Start

Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=mcp-tool-shop.runforge) or download the `.vsix` from [Releases](https://github.com/mcp-tool-shop-org/runforge-vscode/releases).
